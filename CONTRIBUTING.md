# Kontribusi untuk Gibrig

Terima kasih sudah tertarik untuk berkontribusi! Panduan ini membantu Anda memulai.

## Setup Development

### Prerequisites
- Node.js 18+
- Yarn 1.22+

### Langkah-langkah

1. **Fork & Clone**
   ```bash
   git clone https://github.com/YOUR_USERNAME/websitegibrig.git
   cd websitegibrig
   ```

2. **Install Dependencies**
   ```bash
   cd frontend
   yarn install
   ```

3. **Create Feature Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

4. **Development**
   ```bash
   yarn start
   ```

5. **Test**
   ```bash
   yarn test
   yarn build
   ```

## Pull Request

- Keep PRs focused pada satu fitur/fix
- Write clear commit messages (dalam Bahasa Indonesia atau English)
- Update documentation jika diperlukan
- Pastikan build berhasil sebelum submit

## Code Style

- Follow existing code patterns
- Use meaningful variable names
- Add comments untuk logic yang kompleks

## Reporting Issues

- Check existing issues terlebih dahulu
- Provide clear description dan steps to reproduce
- Include screenshots jika relevant
